DESCRIPTION:

The purpose for this website is to allow users to
enter data that I can view and alter in a local
sqlite3 database. The program utilises...
 - Flask for back-end management, 
 - flask-wtforms for data validation and security, 
 - flask-Bootstrap for convenience, and 
 - sqlite3 for database management.

INSTRUCTIONS:

~~ The packages installed via pip are:
    "Flask"
    "Flask-WTF" 
    "flask_bootstrap"
    "email_validator"

    os, datetime, and sqlite3 are all built-in.

 >1. run "create_db.py" to create the database
 >2. run "app.py"
 >3. Go to your browser, and in the address bar go to
     'localhost:5000/'
